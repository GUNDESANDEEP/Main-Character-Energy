# React Components Practice

This project demonstrates the creation of 5 reusable React components: Header, Footer, Card, Button, and Form. Each component implements dynamic rendering using props and state.

## Components

- **Header**: Displays a title with a toggleable menu using state.
- **Footer**: Shows a list of links passed as props.
- **Card**: Renders a card with title, content, and optional image.
- **Button**: A customizable button with text, click handler, and variant styles.
- **Form**: A form with name and email fields, managing state for input values and submission.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.
